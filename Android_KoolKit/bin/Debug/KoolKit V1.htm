KoolKit V1.0.x

 

Android KoolKit is a .net application designed to provide easy access to modifications of several models of Samsung Galaxy S5. This application is a work in progress and changes may be made without any prior notification from the developer.

Instructions

 

NOTE: Watch your device, for each option you choose a notification will display on your device for each step, when no more notifications are being displayed that is an indicator that the procedures have completed.

Bloatware

KoolKit comes with a default configuration file that is keyed to remove the majority of Samsung and AT&T branded apps (Bloatware) this file is located in the application folder inside the KoolKit_V.x.x/bloat folder and is named nobloat.txt.

It also includes the ability to remove the Samsung Knox components however, these are commented out and have not been tested so enable them at your own risk.

To enable items in the list simply remove the # from the beginning of the line, to add an item to the list include the full path and name of the file to be deleted (i.e. /system/app/<appname>.apk), remember to include an item for the corresponding odex file if one exists.

Build.prop

The performance enhancements come with an included default set of additions to the build.prop file those are located in the application folder inside the KoolKit_V.x.x/tweaks folder, to edit this file you simply add the setting you want inside the commented section of the file. Then save the file and run the program to install or update the settings. If you have already applied the settings you will be prompted to update.

Init.d support

Currently this is not user customizable and is created via hard coded instructions within the main executable of the application.

Bootanimation

Currently enabled for Android KitKat 4.4.x additional code may be included to enable this for earlier version of the operating system. The program comes with a default Google bootanimation to include your own or other custom bootanimation copy the bootanimation.zip file into the application folder inside the KoolKit_V.x.x/media folder. This modification will also disable the custom AT&T startup sound. If you wish to have your own Startup audio it must be named PowerOn.ogg and must reside in the /system/media/audio/ui folder.

 

Application Install

From the menu choose apk then choose between installing as a system app (/system/priv-app/) or as a user app, in the file dialog browse to the apk that you want to install and select it.

 

Busybox Install

From the menu choose tools and then choose install busybox
